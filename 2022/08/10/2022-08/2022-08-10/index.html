可靠性定理

（可靠性）. 设 Γ 是公式集，φ 是公式。如果 Γ ⊢ φ，那么 Γ ⊨ φ。

可满足定理

设 Γ 是公式集。称 Γ 是极大一致的（maximally consistent），如果 Γ 是一致的且所有真包含 Γ 的公式集都不是一致的；否则称 Γ 不是极大一致的。

可扩张定理

设 Γ 是 L1 的公式集，∃xφ ∈ Γ 是公式，c 不在 Γ 的任一公式中出现且 φ(c)是自由代入。如果 Γ 是一致的，那么 Γ ∪ {φ(c)} 也是一致的。

无端点稠密线序理论

公理：

  • O1 : ∀x¬(x < x)；
  • O2 : ∀x∀y∀z(x < y ∧ y < z → x < z)；
  • O3 : ∀x∀y(x < y ∨ y < x ∨ x ≃ y)；
  • O4 : ∀x∀y(x < y → ∃z(x < z ∧ z < y))；
  • O5 : ∀x∃y∃z(y < x ∧ x < z)。
